Smart Expense Advisor is a web application built using Java Spring Boot that helps users manage their monthly expenses and savings goals.
The system allows users to register, set a monthly budget, track expenses, and plan savings goals


 Features :-

-> User registration and login

-> Monthly budget management

-> Expense tracking

-> Savings goal planning

-> Dashboard showing remaining budget


Technologies Used

-> Java

-> Spring Boot

-> JDBC

-> Oracle Database

-> HTML

-> CSS
